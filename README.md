# FuzzleWorld

**Fuzzle = felt + puzzle.** FuzzleWorld is a collection of downloadable felt-puzzle kits — build a realistic layered-felt minibeast, take it apart, and build it again, like Lego. Part of [Giant Insect World](https://github.com/).

## How a kit works

Each kit comes with **squares of felt** and a **printable template**:

1. **Print** the template on A4 (landscape) at **100% / actual size** — the piece shapes are true-to-size.
2. **Trace** each shape onto the matching colour of felt.
3. **Cut** out the pieces.
4. **Build** your creature by laying the pieces on the premade felt mat/pouch.
5. **Rebuild** it whenever you like — no glue or stitching needed (optional only).

The mat doubles as a premade pouch to store the pieces and the folded instruction leaflet.

## Structure

- `templates/` — one folder per kit. Each contains:
  - the print-ready template (`*.pdf` / `*.svg`) — an A4 landscape leaflet that folds into a 6×2 panel booklet with a cover, cut pieces at actual size, build steps, and species facts
  - the reference photo(s), credited in the filename
  - `SOURCE.txt` — image licences and attribution

## Files & formats

- **`FuzzleWorld-all-kits.pdf`** — the whole collection as one 19-page booklet (intro + 18 kits). **Print-ready**: text is outlined, so it renders identically on any device/printer.
- **`editable-pdfs/`** — the same pages with **text kept editable** and fonts embedded (better for importing into design tools like Canva). Includes per-kit PDFs and a combined `FuzzleWorld-all-kits-editable.pdf`.
- **Per-kit `.svg`** (in each kit folder) — the **editable vector source**. Best for editing shapes/colours (e.g. Canva SVG upload / Magic layers). Fonts used: **Nunito** (headings) and **Source Serif 4** (body) — both free; add them in your editor for an exact match.
- **Per-kit `.pdf`** (in each kit folder) — the individual print-ready sheet (outlined).

Print any sheet at **100% / actual size** so the felt pieces come out true-to-size.
